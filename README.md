# Blog App Backend

Esse é o projeto do backend do blog que está sendo desenvolvido na disciplina de Progrmação Web do IFBA SAJ.

## Como rodar o projeto

No terminal dentro do projeto, execute

```sh
mvn clean install spring-boot:run
```
